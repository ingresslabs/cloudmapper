cloudmapper

A standalone Rust CLI that exports infrastructure reality into agent-readable SQLite, JSONL, and graph artifacts. Scan AWS or Kubernetes, import Terraform state, compare drift, and inspect the result in a local Cytoscape graph UI.

One local map for scanners, Terraform, and agents.

cloudmapper writes a portable bundle with inventory, graph facts, compare findings, and a queryable SQLite store. The format is designed for direct inspection by humans and structured context loading by AI agents.

AWS + Kubernetes Scan cloud accounts or Kubernetes contexts into the same resource and relationship model.
SQLite map.db Persist normalized resources, graph edges, Terraform mappings, and findings for local queries.
Drift findings Compare scan reality against Terraform state and keep evidence with blast radius context.
Agent export Emit one portable JSON artifact with redacted context ready for agent workflows.

Cytoscape graph demos.

The built-in UI renders AWS and Kubernetes demo bundles without external services. Generate the bundles locally with make demo or make demo-k8s.
